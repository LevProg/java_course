Описание структуры закодированного файла
1. Общая информация:
Закодированный файл содержит две части:
Словарь частот.
Закодированные данные.

2. Формат файла:
Размер словаря :
Целое число (int), указывающее количество уникальных символов.
Словарь частот :
Для каждого символа:
Символ (char) — 2 байта.
Частота символа (int) — 4 байта.
Длина бинарных данных :
Целое число (int), указывающее длину бинарных данных в битах.
Бинарные данные :
Последовательность байтов, содержащих закодированные данные.
3. Пример:
Исходный текст: "abracadabra"

Частоты символов :
a: 5
b: 2
r: 2
c: 1
d: 1
Коды Хаффмана :
a -> 0
b -> 10
r -> 110
c -> 1110
d -> 1111
Закодированные данные :
01011001110011110101100
Формат файла :
Размер словаря: 5
Словарь:
  'a' -> 5
  'b' -> 2
  'r' -> 2
  'c' -> 1
  'd' -> 1
Длина бинарных данных: 23 бита
Бинарные данные:
  01011001 11001111 0101100
